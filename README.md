# Shreyas Dorigol — Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript to showcase skills, projects, and resume.

## 📁 Project Structure

```
portfolio/
└── website/
    ├── index.html          # Main portfolio page
    ├── project1.html       # Digital Interface for Department Operations
    ├── project2.html       # Personal Portfolio Website
    ├── project3.html       # Online Learning Platform
    ├── styles.css          # Shared stylesheet
    ├── assets/
    │   └── Shreyas-resume.pdf
    └── images/
        ├── profile.png
        └── projects/
            ├── projects1.jpeg
            ├── projects2.jpeg
            ├── projects3.jpeg
            ├── projects4.png
            └── projects5.png
```

## ✨ Features

- **Sticky navigation bar** with smooth section links
- **Dark mode toggle** (persists within the session)
- **Responsive project cards** with hover animations
- **Downloadable resume** linked directly from the navbar
- **Contact section** with Email, LinkedIn, and GitHub links

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)
- MySQL *(used in project backends)*

## 📌 Sections

| Section  | Description                                      |
|----------|--------------------------------------------------|
| Home     | Profile photo, name, and headline                |
| About    | Brief bio and background                         |
| Skills   | HTML/CSS/JS, Java, MySQL                         |
| Projects | Three featured projects with detail pages        |
| Resume   | Opens `Shreyas-resume.pdf` in a new tab          |
| Contact  | Links to email, LinkedIn, and GitHub             |
